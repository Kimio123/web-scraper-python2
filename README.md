# Web Price Scraper Pro (Python)

Este es un extractor de datos automatizado que no solo recopila información, sino que genera reportes de nivel ejecutivo listos para su uso inmediato.

# Funcionalidades Avanzadas
* **Extracción Inteligente:** Captura datos en tiempo real de catálogos web.
* **Limpieza Automática:** Elimina símbolos extraños y normaliza formatos de moneda.
* **Diseño de Excel "One-Click":** Genera un archivo `.xlsx` con:
    * Encabezados resaltados en verde corporativo y texto en negrita.
    * Ajuste automático de ancho de columnas (Auto-fit) según el contenido.
* **Apertura Automática:** El sistema abre el reporte apenas termina la ejecución para una experiencia de usuario fluida.

# Tecnologías
* **Python 3.x**
* **Requests & BeautifulSoup4** (Web Scraping)
* **Pandas & Openpyxl** (Gestión y Diseño de Excel)

# Instalación rápida
1. Clona el repo.
2. Instala dependencias: `pip install -r requirements.txt`
3. Ejecuta: `python scraper.py`
